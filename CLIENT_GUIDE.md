## 配置方法

### 方法一: 扫描二维码（推荐）
1. 在 3x-ui 管理界面获取用户的二维码
2. 使用客户端扫描功能导入配置
3. 选择节点并连接

### 方法二: 复制配置链接
1. 复制 3x-ui 生成的配置链接
2. 在客户端中选择 "从剪贴板导入"
3. 测试连接

### 方法三: 手动配置
如果自动导入失败，可以手动填写配置信息：

#### VMess 配置参数
- **服务器地址**: 你的域名
- **端口**: 443
- **用户ID**: 从 3x-ui 复制
- **传输协议**: ws (WebSocket)
- **路径**: /vmessws
- **TLS**: 启用
- **伪装域名**: 你的域名

#### VLESS 配置参数  
- **服务器地址**: 你的域名
- **端口**: 8443
- **用户ID**: 从 3x-ui 复制
- **传输协议**: ws (WebSocket)
- **路径**: /vlessws
- **TLS**: 启用
- **流控**: 留空

## 客户端详细配置

### v2rayN (Windows)

1. **下载安装**
   - 从 GitHub 下载最新版本
   - 解压到任意目录
   - 运行 v2rayN.exe

2. **导入配置**
   - 右键系统托盘图标
   - 选择 "从剪贴板导入批量URL"
   - 或扫描二维码

3. **连接测试**
   - 选择服务器节点
   - 右键 > "测试服务器延迟"
   - 设置为活跃服务器

### v2rayNG (Android)

1. **安装应用**
   - 从 Google Play 或 GitHub 下载
   - 安装 APK 文件

2. **配置导入**
   - 点击右上角 "+" 
   - 选择 "从剪贴板导入"
   - 或扫描二维码

3. **连接设置**
   - 选择配置文件
   - 点击右下角连接按钮
   - 允许 VPN 连接

### Shadowrocket (iOS)

1. **添加节点**
   - 点击右上角 "+"
   - 扫描二维码或输入配置

2. **连接设置**  
   - 选择节点
   - 点击连接开关
   - 允许 VPN 配置

## 连接优化

### 1. 协议选择建议
- **日常使用**: VMess + WebSocket + TLS
- **高速下载**: VLESS + TCP + TLS  
- **严格环境**: Trojan + TLS

### 2. 客户端优化
- 启用 "Mux" 多路复用（如果支持）
- 调整并发连接数
- 选择合适的路由规则

### 3. 网络优化
- 使用 UDP 加速（如果支持）
- 开启 TCP Fast Open
- 调整 MTU 大小

## 故障排除

### 连接失败
1. **检查服务器状态**
   - 确认 3x-ui 服务运行正常
   - 检查端口是否开放

2. **验证配置信息**
   - 确认域名解析正确
   - 检查端口和路径设置
   - 验证用户ID正确

3. **网络诊断**
   - 尝试不同的连接端口
   - 测试直连IP地址
   - 检查本地防火墙设置

### 速度慢
1. **更换协议**
   - 尝试 VLESS 协议
   - 使用 TCP 传输方式
   - 关闭不必要的加密

2. **调整客户端**
   - 增加并发连接数
   - 启用多路复用
   - 选择合适的路由模式

### 无法访问特定网站
1. **检查路由规则**
   - 确认目标网站在代理列表中
   - 调整客户端路由设置
   - 尝试全局代理模式

2. **DNS 设置**
   - 使用代理服务器DNS
   - 或使用公共DNS（8.8.8.8）

## 使用技巧

### 1. 多配置管理
- 为不同用途创建不同配置
- 使用配置组功能
- 定期更新和备份配置

### 2. 流量节省
- 使用分流规则
- 仅代理需要的流量
- 避免全局代理模式

### 3. 安全建议
- 定期更新客户端
- 不要分享个人配置
- 使用复杂的连接密码
